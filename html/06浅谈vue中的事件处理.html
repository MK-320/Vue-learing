以上的$event 再无参传入时可以省略，因为默认会传入一个$event事件 $event这种写法是vue特有的这么写

总结：1. 使用 v-on:xxx 或者 @: xxx 来绑定事件
2. 事件的回调 需要配置在 methods中 ，而最终会在 vm 对象中
3. methods中 配置的函数 不要用箭头函数 ，否则 this 就不是vm 了
4.methods 中配置的函数都是 vue 管理的函数 ，this 指向的是 vm 或者组件实例对象
5. @click ="fun" 和 @click ="fun($event)" 写法一样，只不过前者省略了 event ，而且后者可以传入参数
